# conversion-utils

--------------------------------------------------------------

	requirements:
	
		> pip install pdf2docx, docx2pdf
		
		
--------------------------------------------------------------

	pdf2word.py :   conversione .pdf -> .docx

		converte da .pdf a .docx

		utilizzo:  > from pdf2word import converti
				   > converti("C:\Users\ITMACOS\pippo.pdf")

		oppure:    > py pdf2word.py C:\Users\ITMACOS\pippo.pdf

--------------------------------------------------------------

	word2pdf.py : conversione .docxpdf -> .pdf

		converte da .docx a .pdf

		utilizzo:  > from word2pdf import converti
				   > converti("C:\Users\ITMACOS\pippo.docx")

		oppure:    > py word2pdf.py C:\Users\ITMACOS\pippo.docx

--------------------------------------------------------------

	oppure usare l'interfaccia a finestre lanciando

		   > py ./interface.py
